# Steam-Key-Tumbler
<p align="center">
Receive steam keys by personal share link<br>
(based on Vuepack which uses Vue 2, Vuex, Vue-router and Webpack 2)
</p>


Installation:
```bash
$ yarn install
```

Start development server:
```bash
$ yarn run dev
```

Build for production:
```bash
$ yarn run build
```

Lint you code:
```bash
$ yarn run lint
```
